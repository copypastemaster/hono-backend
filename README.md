# Hono backend starter template

This is a basic backend template that has sufficient needs to start a project right away.

It handles the database setup, some configs, file structures so you can start writing the logics right away.

### This project uses:
- hono
- bun
- drizzle ORM

### NOTE: This is note configured for cloudflare-workers

To install dependencies:
```sh
bun install
```

To run:
```sh
bun run dev
```

open http://localhost:3000
